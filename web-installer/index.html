Install FluidMonitor

A touch DRO monitor for FluidNC over ESP-NOW. Flash it onto a supported ESP32 Cheap Yellow Display directly from the browser.

This browser doesn't support Web Serial. Open this page in Chrome or Edge on a desktop computer. Web Serial needs a secure connection. Use the HTTPS link, not a local file.

Checking latest release… · Ensure a supported CYD is connected via USB

The installer performs a clean flash for reliable setup. Saved settings on the device may be reset.

Requires FluidNC with ESP-NOW. This isn't in an official FluidNC release yet - until it's merged upstream, your controller must run the feature/esp-now branch (firmware below).

FluidNC firmware

ESP-NOW isn't in an official FluidNC release yet. Until it's merged, flash one of these firmware.bin builds of the feature/esp-now branch onto your controller - through the FluidNC web UI (OTA update), or by building from source.

  1. Connect

    Plug the CYD into your computer with a data-capable USB cable.

  2. Install

    Click Install, choose the ESP32 serial port, then wait for the flash to finish. On resistive boards, tap the on-screen target when prompted.

  3. Pair

    Enable ESP-NOW on your FluidNC controller, then open the Settings tab and tap Pair to link them.
